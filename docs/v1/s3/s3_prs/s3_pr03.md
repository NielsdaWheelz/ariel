# PR-03: Grounding Safety Hardening

## Goal

Close remaining Slice 3 grounding safety gaps so externally grounded answers stay citation-gated and uncertainty-first under conflicting evidence and mixed proposal sets.

## Builds On

PR-02 merged.

## Acceptance

- when retrieval evidence conflicts for the same factual claim, Ariel does not present a definitive claim and instead returns explicit uncertainty plus a concrete recovery step.
- turns that execute `cap.search.web`, `cap.search.news`, and/or `cap.weather.forecast` return grounded synthesis with inline citations and synchronized `assistant.sources[]`, including turns that also include non-retrieval proposals.
- surfaced assistant text does not present external factual assertions unless those assertions are backed by cited provenance artifacts; unsupported assertions are replaced with uncertainty language.
- non-retrieval proposal outcomes remain inspectable via structured turn lifecycle/event surfaces in the same response flow, without regressing action lifecycle/event auditability.
- assistant message content remains grounded narrative only for retrieval-backed turns and does not revert to raw action-result appendix text for mixed proposal sets.
- partial or failed retrieval within grounded turns remains explicitly disclosed and recoverable without weakening citation-gating guarantees.
- regression coverage adds explicit conflict-evidence and mixed-turn citation-gating scenarios and blocks release on failure.

## Non-Goals

- does not add a full claim-extraction, contradiction-detection, or NLI platform.
- does not add cross-turn truth maintenance or long-horizon fact reconciliation.
- does not add ranking/relevance ML changes beyond deterministic MVP heuristics.
- does not expand retrieval scope beyond existing web/news/weather read capabilities.

## Key Decisions (if any beyond L2)

**Grounded narrative / telemetry separation**: if grounded retrieval executes in a turn, `assistant.message` stays citation-gated user narrative; execution telemetry (including non-retrieval outcomes) stays in structured lifecycle/event surfaces.

**Conflict handling is fail-closed**: unresolved evidence disagreement is treated as uncertainty with recovery guidance, never as a definitive externally grounded fact.
